# test soundcloud player component

forked from https://github.com/raoulus/react-webpack-template

This project includes:
- React 15.4.2
- webpack 2.2.1
- express 4.14.1
- babel loader 6.3.2 and babel core 6.23.1
- livereload 0.6.0

## Getting started

```bash
# install
npm install

# build client
npm run build

# start server
npm start

# start server + watch for changes
npm run dev
```
